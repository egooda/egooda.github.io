Devanescente
Editorial - Graphic Design
 
 
O projeto Devanescente é o meu trabalho de conclusão de curso em Design pela UNESP, neste projeto quis experimentar mais na parte da diagramação na tentativa de sair um pouco do que tradicionalmente faço, assim pude explorar algumas novas técnicas em principal por se tratar de um projeto bastante pessoal.
 

 
Devanescente is my final graduation project in Design, in this project I wanted to experiment more at diagramation trying to go out of my safe zone, so I could explore som new techniques mainly by this being an very personal project.

This material showed below is not the whole project, only the Report of it, the whole project can be found in the website at the end of the images.
 
 
You can see the full project in the following link
http://miscnmess.tumblr.com
